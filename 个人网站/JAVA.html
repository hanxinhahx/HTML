JAVA教程

JAVA简介

  Java是由Sun Microsystems公司于1995年5月推出的Java面向对象程序设计语言和Java平台的总称。由James Gosling和同事们共同研发，并在1995年正式推出。


  JavaSE（J2SE）（Java2 Platform Standard Edition，java平台标准版）

  JavaEE(J2EE)(Java 2 Platform,Enterprise Edition，java平台企业版)

  JavaME(J2ME)(Java 2 Platform Micro Edition，java平台微型版)。

  2005年6月，JavaOne大会召开，SUN公司公开Java SE 6。此时，Java的各种版本已经更名以取消其中的数字"2"：J2EE更名为Java EE, J2SE更名为Java SE，J2ME更名为Java ME。

基础语法

结构简介

接下来就正式进入JAVA编程的学习

  一个 Java 程序可以认为是一系列对象的集合，而这些对象通过调用彼此的方法来协同工作，将在面向对象这一模块中详细将解。下面我们来重点介绍一些结构以及数组。

  首先我们要明白JAVA中的语句都要放在一个类中，这相当于一个容器，而代码就是装在这里面的，当然，如果你之前学过C语言或者C++的话，你就可以把JAVA中的类理解成C++的函数，也可以将方法说成函数。在一般情况下，程序的执行但是从主方法开始的。因此，再一个类中只可能有一个主方法或者没有主方法（这种情况后面再讲）。

下面还是以HelloWorld这个程序来说明

package BasicKnowge;
public class HelloWorld {
public static void main(String[] args) {
System.out.println("HelloWorld!");
  }
}

  package是包，你可以将它理解成是一个项目，这个项目里会由许许多多的类组成，它就是拿来装类的。第二行是我们创建好的类，然后在我们的类中有主方法 public static void main(String[] args){……}这个结构是固定的，现在只需你记住即可。主方法中是我们要输出的语句。当然，一个类中不太可能只有一个主方法，它还拥有其他方法以及成员变量，在接下来的教程中我们也会详细说明

  注意：在每句语句结束后要加一个英文状态下的;，还有一点，当输出语句过长时，你可以将其拆成好几部分，用""包裹，进行分行，中间用+号连接。

数据类型

  计算机本质上是对不同的数据进行处理，而数据又分为多种类型，因此，我们有必要先去了解JAVA中的数据类型，这是接下来学习的重要基础。

  其中8种基本数据类型是JAVA内嵌的，在任何操作系统中都具有相同的大小和属性，而引用数据类型是编写程序的人员中间定义的。


下面就来详细地看看各个数据类型

1.整数类型（整型）

  整数类型数据类型是用来存储整数数值的，没有小数部分根据存储的数据大小以及所占存储空间大小可分为以下4种：byte、short、int、long。具体数值范围见下表

整数类型
类型名 占用空间 取值范围
byte 8位（1个字节） -27~27-1
short 16位（2个字节） -215~215-1
int 32位（4个字节） -231~231
long 64位（8个字节） -263~263-1

2.浮点型数据类型

  浮点型数据类型是存储小数数值，分为float和double，double的精度更高，如下图：

浮点型数据类型
类型名 占用空间 取值范围
float 32位（4个字节） 1.4E-45~3.4E+38,-3.4E+38~-1.42E-45
double 64位（8个字节） 4.9E-324~1.7E+308,-1.7E+308~-4.9E-324

3.字符数据类型

  字符类型用于存储一个单一的字符（不是字符串），在JAVA中用char表示，每个char类型的变量都会站两个字节，需要用一个英文半角格式的单引号括起来，实质上是存储将字符的ASCⅡ码，范围为0~65535，也可以直接把赋给char类型变量一个整数，表示字符的ASCⅡ码。

4.布尔数据类型

  布尔数据类型用来存储布尔值，在JAVA中用boolean表示，有true和false两个值。

变量类型转换

自动类型转换

  自动类型转换又叫隐式类型转换，指的是两种数据类型在转换的过程中不需要进行显式地进行申明。要实现自动类型转换，必须用时满足两个条件：①两种数据类型彼此兼容。②转换后的数据类型范围要大于原类型。

  可以进行自动类型转换的情况如下：

  1.整数类型之间可以实现转换。如byte类型的数据可以转换成short、int、long类型的变量，short、char类型数据可以赋值给int、long类型，int可以赋值给long类型。

  2.整型类型可以转换成float类型，如nyte、short、int、char类型的变量可以赋值给float类型的变量。

  3.除double自身以外，其他几种数据类型都可以转换成double类型变量。

强制类型转换

  强制类型转换也称为显式类型转换，指的是两种数据转换需要进行显式的声明。当两种数据类型彼此不兼容，或者范围小于要转的数据类型，这时就需要强制类型转换。

  例：将long类型转换成int

  long a;

  int b=(int)a;

  目标类型 变量名=（目标类型）值;

变量作用域

  变量要定义之后才可以使用，但是并不意味着定义好的变量可以一直使用，它也有自己的作用范围，这个范围就是作用域。在一个程序中在一个达括号中定义变量，则这个变量的作用域就在这个范围之内，且在变量定义语句之后，超出这个范围就不能使用该变量。

运算符


1.算术运算符

  算术运算符是JAVA中用于处理四则运算的运算符，接下来看下面一张表格来解释。

算术运算符
运算符 运算 范例 结果
+ 正号 +3 3
- 负号 b=4;-b -4
+ 加 5+5 10
- 减 5-5 0
* 乘 5*5 25
/ 除 5/6 1
% 取模（取余数） 7%5 2
++ 自增（前） a=2;b=++a; a=3;b=3
++ 自增（后） a=2;b=a++; a=3;b=2
-- 自减（前） a=2;b=--a; a=1;b=1
-- 自减（后） a=2;b=a-- a=1;b=1

  注意：自增和自减符号不同位置结果也不同，可以认为：符号在变量前，则先进行自增/自减，然后参与其他运算，而符号在后，则先进行其他运算，再进行自增/自减运算。

2.赋值运算符

  赋值运算符是将常量变量、表达式的值赋给某个变量。见下表。

赋值运算符
运算符 运算 范例 结果
= 赋值 a=3;b=2 a=3;b=2
+= 加等于 a=3;b=2;a+=b a=5;;b=2
-= 减等于 a=3;b=2;a-=b a=1;b=2
*= 乘等于 a=3;b=2;a*=b a=6;b=2
/= 除等于 a=3;b=2;a/=b a=1;b=2
%= 模等于 a=3;b=2;a%=b a=1;b=2

  注意：赋值过程中，运算顺序从右向左，将右边表达式的结果赋给左边变量。需要注意以下两点：

   1.在java中可以通过一句语句给多个变量进行赋值，如“x=y=z=1”

   2.赋值运算符的优先级较低，我们在之后的运算符优先级中会详细讲到。所以，应先进行其他运算，最后再进行赋值运算。几个特殊的赋值运算符也遵守这样的规则，例如“a+=b”，它的意思为“a=a+b”，其他几个也同理。

3.比较运算符
